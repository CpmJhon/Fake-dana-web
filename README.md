# 📌 FaceCheck Attendance System

Website ini adalah aplikasi absensi karyawan berbasis web yang menggunakan kamera untuk melakukan check-in dan check-out dengan foto sebagai bukti kehadiran. Sistem ini berjalan sepenuhnya di browser tanpa backend (serverless) dengan penyimpanan data menggunakan IndexedDB.

# 🚀 Features

✨ **Smart Attendance**

* 📥 Check-in (foto + waktu)
* 📤 Check-out (foto + waktu)
* ⏱️ Auto real-time clock

📸 **Camera System**

* Live camera capture
* Preview foto sebelum save

🖼️ **Fallback Upload**

* Upload foto dari device jika kamera gagal

🗂️ **Offline Database**

* IndexedDB (100% local)
* No internet needed

📊 **Data Table**

* Nama, tanggal, jam
* Durasi kerja
* Foto masuk & pulang

📅 **Filter System**

* Filter by date
* Reset filter instant

📄 **Export PDF**

* Full report
* Filtered report

🌙 **Dark Mode UI**

* Clean modern interface
* Toggle mode

# 🌐 GitHub Pages (Landing Page Setup)

## 🔥 Cara Deploy

### 1. Upload ke GitHub

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/CpmJhon/Absen-Karyawan-Web.git
git push -u origin main
```

---

### 2. Aktifkan GitHub Pages

* Masuk ke:
  **Settings → Pages**
* Pilih:

  * Source: `main branch`
  * Folder: `/root`
* Klik **Save**

---

### 3. Akses Website

Website kamu akan live di:

```
[https://github.com/CpmJhon/Absen-Karyawan-Web.git]
```
